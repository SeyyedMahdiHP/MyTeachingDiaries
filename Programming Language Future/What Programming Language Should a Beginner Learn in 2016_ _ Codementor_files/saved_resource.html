                          "However, JavaScript is also known to be a difficult language as it is untyped and thus is difficult to debug."

                          "What language should a coding beginner learn? Clearly, if you wish to get into web development, youd inevitably have to learn JavaScript, whether you like it or not."

                          Sorry, Ms. Chen, that isn't right. JavaScript is a dynamically-typed language like Python, Ruby, Smalltalk and others. It is not "untyped." Python, Ruby and Smalltalk are not difficult languages at all. They've been used very successfully to write large, complex applications. Some Python applications are over a quarter of a million lines long. Smalltalk has been used to write a million-line battle simulation program for the U.S. joint military called JWARS.

                          The reason why JavaScript is "difficult" and "difficult to debug" is because it is intrinsically a deeply-flawed programming language. And the reason it is apparently so popular is because much of the IT community has bought into its insane hype.

                          If you're really interested in web development, I strongly recommend using a transpiled language and learning just enough JavaScript to interface with jQuery. I use Amber Smalltalk myself, but you can choose Python, Dart, Clojure, Scala, whatever. Use anything but JavaScript!

                          Otherwise, this is a very excellent article. Congratulations.
